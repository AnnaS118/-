# -# Expense Tracker (Трекер расходов)

**Автор:** Самайсак Анна

## Описание программы

Expense Tracker — это графическое приложение для отслеживания личных расходов. Программа позволяет добавлять расходы с указанием суммы, категории и даты, фильтровать записи по категории и дате, а также автоматически подсчитывать общую сумму расходов за выбранный период. Данные сохраняются в файл `expenses.json` в формате JSON.
